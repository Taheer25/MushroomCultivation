🍄 Mushroom Classification Using Machine Learning

📌 Overview

This project aims to classify mushrooms as edible or poisonous using machine learning techniques and structured dataset analysis.

It demonstrates practical application of data science in solving real-world classification problems.

---

⚙️ Key Features

- Binary classification model for mushrooms
- Data preprocessing and feature engineering
- Exploratory Data Analysis (EDA)
- Model evaluation with multiple metrics
- Visualization of insights

---

🧠 Approach

- Cleaned and processed dataset using Pandas and NumPy
- Applied Logistic Regression algorithm
- Performed feature selection and transformation
- Evaluated using accuracy, precision, recall, and confusion matrix

---

📊 Results

- Achieved 92% accuracy on classification model
- Trained and tested on dataset of 8000+ samples
- Demonstrated reliable prediction of edible vs poisonous mushrooms

---

🛠️ Tech Stack

Python, NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn, Jupyter Notebook
